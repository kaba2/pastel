Filters
=======

[[Parent]]: pasteldsp.txt

In Pastel, a _filter_ is a piecewise continuous function ''RR -> RR'' 
with bounded support.

Practice
--------

The Filter class is an abstract base class for 
a 1-dimensional reconstruction filter. It is defined as:

[[CppCode]]:
	class Filter
		: public ReferenceCounted
	{
	public:
		Filter(real radius,
			const std::string& name);
		virtual ~Filter();

		virtual real evaluateInRange(real position) const = 0;

		void setRadius(real radius);
		real radius() const;

		real evaluate(real position) const;
		const std::string& name() const;
	};
